Написал две утилиты: для создания файла и для сортировки файла.

В создании файла предусмотрены дубликаты строк.
Создание файлов не быстрое, так как каждый символ генерируется случайно. 50 Гб будет генерироваться минут 40.

Сортировка проиисходит следующим образом:

1) Файл делится на маленькие части и эти части одновременно сортируются. Всё это происходит в папке /sortarea
2) Маленькие части постепенно сливаются в один большой файл.
